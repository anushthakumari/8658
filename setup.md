# Setup Instructions

1. Extract the zip file
2. Open terminal in the project root
3. Setup Backend:
   cd Backend
   npm install
   npm run build
   
4. Setup Frontend:
   cd ../Frontend
   npm install
   npm run build

5. Start the application:
   # Terminal 1 (Backend)
   cd Backend && npm run dev
   
   # Terminal 2 (Frontend)  
   cd Frontend && npm run dev
